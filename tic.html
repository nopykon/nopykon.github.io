TIC-80 is a fantasy console with artificial limitations similar to PICO-8. 16-colors, 240x136 and very limited space for code and assets.

Had an idea I for lighting I wanted to try out. It achieves a cel-shaded look by clipping polygons (when need be) by brightness. (No shaders and only 16-colors with TIC-80).


I also wanted to try out on-the-fly "tesselation". Earlier lighting model is used here with positional lighting (and normals ignored). (Normals are still used by the... vertex extrapolation function in the tesselation.)



My first fantasy console demo, made in 2017 for FCGJ#1. This demo was fancy for a while, because it was made before TIC-80 had the function textri(). You can try it out here, but you're best off just watching the gif, because that's all there is to the demo anyways.
